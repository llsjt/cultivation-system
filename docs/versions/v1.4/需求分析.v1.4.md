版本：v1.4

## 已完成内容总览

v1.4 记录自用 MVP 已完成范围，作为后续版本对照基线。范围重点是本地保存学习项目、资料、出关记录和待记录会话，并在首页恢复最应该继续的资料。

| 阶段 | 已完成内容 | 验收/证据 |
| --- | --- | --- |
| M0 本地数据骨架 | Electron + React + SQLite 应用骨架；核心表；Drizzle 迁移；外键、CHECK、枚举约束；DTO 与 IPC 命令契约；主进程 Service 基础能力 | 数据库约束、迁移和基础命令由测试覆盖 |
| M1 项目和资料基础管理 | 项目创建、编辑、删除、列表；资料创建、编辑、删除、详情；删除确认；表单 trim 与基础校验；空状态 | UI 与 service 回归覆盖基础 CRUD |
| M2 学习进度展示与推荐 | 项目进度计算；首页推荐可继续资料；首页、项目详情、资料详情进度一致展示；暂停/完成资料不进入推荐 | Service 测试覆盖推荐排序、进度一致性与状态派生 |
| M2.5 手动出关闭环 | `record_only` 与手动出关记录；生成 `study_logs`；保存后刷新首页/详情；关闭重开恢复进度、下一步和日志 | 冒烟流程覆盖保存学习记录和重启恢复 |
| M3 继续闭关与待记录 | 文件/文件夹/URL 安全打开；风险确认；待记录会话创建、恢复、冲突处理与放弃；`last_opened_at` 与 `last_studied_at` 区分 | Service 测试覆盖 pending、冲突、删除清理、URL/路径安全与事务一致性 |
| M4 主题与反馈 | 深色修仙主题；双层可理解文案；保存成功、进度变化、100% 完成反馈；Toast 队列；基础可访问性与焦点处理 | UI 验收覆盖关键路径、Toast、focus trap 和 Escape 行为 |
| M5 自用验收与打包 | 自用核心路径回归；portable 打包；本地优先验证；关闭重开冒烟 | 打包、smoke、UI 验收脚本通过或记录风险 |
